# Restock 'n' Roll

A Low Stock Alert Mobile App for Small Businesses.

This repository contains two main parts:

- `backend/` — Node.js + Express API server
- `mobile/` — React Native (Expo) mobile app

Getting started

1. Backend
   - cd backend
   - npm install
   - npm run dev

2. Mobile
   - cd mobile
   - npm install
   - npx expo start

See each subfolder README for more details.
